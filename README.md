# Toposel WordPress Assignment

This project recreates the mobile homepage of an e-commerce website using WordPress and ACF.

## Features
- Dynamic announcement bar
- Hero section with image, heading, subheading & CTA
- Brand logos editable via CMS
- New Arrivals section with 2 product cards
- No hardcoded text or images

## Tech Stack
- WordPress
- PHP
- ACF (Free)
- HTML / CSS

## Setup
- Developed using LocalWP
- Custom WordPress theme
- All content managed via WordPress Admin for non-technical users

## Notes
ACF Free was used to avoid paid plugin dependency while keeping the CMS fully dynamic.
